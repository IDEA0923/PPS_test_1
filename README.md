для запуска нужен токен с openrouter.ai  и  токен с botfather ( оба токена поместить в config.py), после: 

    sudo docker-compose up --build

токены поместить в значения в config.py ( в кавычки )

архитектура : 

    2 контейнера , один с postgres , другой с ботом 
    сначала поднимается postgres , потом надо подождать какое-то время и поднимится бот (таймер на 5с , тк до этого быстрее поднималась БД , но надо ждать чуть дольше )

    бот : парсинг video.json , добавление в postgres 
    далее ждет сообщения от пользователя , отправляет его ИИ , получает ответ ( чуть фильтрует ) , отправляет запрос к БД , отдает пользователю 

LLM: 

    подключение через openrouter.ai 
    промпт - в config.py 


PS : если в логах docker-compose postgres "поднялся" , а bot - нет (должно быть сообщение логгирования ) , то надо подождать , оно "живое" , просто проблема с таймером
